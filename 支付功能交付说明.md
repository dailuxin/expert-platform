# 专家平台 - 支付功能交付说明

## 启动服务器

```powershell
# 1. 杀掉旧进程（如有）
Get-Process -Name node -ErrorAction SilentlyContinue | ForEach-Object { Stop-Process -Id $_.Id -Force }

# 2. 启动（必须用完整路径）
Start-Process -FilePath "C:\Program Files\QClaw\v0.2.22.518\resources\node\node.exe" `
  -ArgumentList "app.js" `
  -WorkingDirectory "C:\Users\dailu\Desktop\expert-platform" `
  -WindowStyle Hidden

# 3. 等待 3 秒，然后访问
# http://localhost:3000
```

默认管理员：`admin` / `admin2026`

## 快速测试流程

### 第一步：准备专家账号
1. 注册一个账号（如 `expert001`）
2. 点击「填写专家资料」，填写行业和简介，提交
3. 用管理员账号登录 → 管理后台 → 待审核 → 点击「通过」
4. 退出管理员，重新登录 `expert001`，此时可以看到「文章」入口，说明专家已通过

### 第二步：普通用户预约并支付
1. 注册第二个账号（如 `user001`）
2. 首页搜索专家，点进去，点「预约咨询」
3. 选择日期、时间、时长，提交
4. 弹出支付窗口，点「模拟支付」
5. 支付成功，订单状态变为「已支付」

### 第三步：专家确认
1. 登录 `expert001`（专家账号）
2. 「我的订单」里看到待确认订单
3. 点「确认预约」→ 状态变为「已确认」
4. 点「标记完成」→ 状态变为「已完成」

### 第四步：退款（可选）
1. 登录 `user001`
2. 「我的订单」→ 点「申请退款」
3. 管理员登录 → 管理后台 → 退款管理 → 通过/驳回

## 如遇问题
- 白屏：按 F12 查看 Console 有无 JS 报错
- 401 未登录：检查 Cookie 是否正常，或重新登录
- 端口被占用：`netstat -ano | findstr :3000` 找到 PID 后 `taskkill /PID <id> /F`
